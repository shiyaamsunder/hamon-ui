# Hamon UI - A minimal React component library.

## How to install

```bash
npm install hamon-ui @stitches/react
```

---

## Import

```jsx
import { Button } from "hamon-ui";
```

## Usage

```jsx
<Button colorScheme="purple">Button</Button>
```

## Components

Components that are available to work with.

- Badge
- Button
- ButtonGroup
- Box
